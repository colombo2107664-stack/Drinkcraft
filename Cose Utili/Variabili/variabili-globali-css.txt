Queste sono le variabili globali CSS del progetto. Per ciascuna variabile, considera i seguenti possibili utilizzi, così posso capire subito dove usarla.

VARIABILI GLOBALI CSS
Definite in: CSS/generale.css

SPAZIATURE

--spazio-3xs
Valore: clamp(0.25rem, 0.5vw, 0.5rem)
Possibili utilizzi:
- piccoli padding interni
- piccoli margini
- distanza minima tra icona e testo
- spaziature molto compatte nei bottoni
- micro-gap tra elementi vicini

--spazio-2xs
Valore: clamp(0.5rem, 0.9vw, 0.75rem)
Possibili utilizzi:
- spazio tra label e input
- piccoli gap verticali
- padding ridotto di chip o badge
- distanza tra elementi secondari
- margini contenuti in box piccoli

--spazio-xs
Valore: clamp(0.625rem, 1.1vw, 0.875rem)
Possibili utilizzi:
- padding interno di piccoli contenitori
- gap tra elementi di navigazione
- distanza tra icona e testo nei pulsanti
- piccoli margini tra righe di contenuto
- spaziatura interna di card compatte

--spazio-sm
Valore: clamp(0.75rem, 1.3vw, 1rem)
Possibili utilizzi:
- gap standard tra elementi vicini
- padding di pulsanti
- margini piccoli ma frequenti
- distanza tra campi di form
- spaziatura interna di box compatti

--spazio-md
Valore: clamp(0.875rem, 1.6vw, 1.125rem)
Possibili utilizzi:
- padding medio di card
- gap tra blocchi nello stesso contenitore
- distanza tra titolo e contenuto
- spaziature tipiche di form e sezioni
- separazione equilibrata tra elementi UI

--spazio-lg
Valore: clamp(1rem, 2vw, 1.5rem)
Possibili utilizzi:
- padding ampio di contenitori
- margine tra sezioni interne
- distanza tra gruppi di elementi
- spaziatura attorno a card importanti
- separazione più ariosa nei layout

--spazio-xl
Valore: clamp(1.25rem, 2.8vw, 2rem)
Possibili utilizzi:
- spazio tra sezioni principali
- padding interno di box importanti
- margini esterni di contenitori grandi
- distanza tra blocchi ben distinti
- respiro extra in aree centrali della pagina

--spazio-2xl
Valore: clamp(1.5rem, 3.5vw, 2.5rem)
Possibili utilizzi:
- distacco tra sezioni grandi
- padding molto ampio
- spaziature generose nei layout desktop
- separazione forte tra blocchi principali
- margini verticali importanti

--spazio-3xl
Valore: clamp(1.75rem, 4vw, 3rem)
Possibili utilizzi:
- spazio tra macro-sezioni
- padding molto abbondante di contenitori evidenti
- hero section o box grandi
- separazione visiva molto forte
- aree che devono respirare molto

- select di dimensione ridotta
- controlli form secondari
- campi brevi

- profili medi
- icone profilo in aree più importanti

--dimensione-icona-badge

- elementi grafici centrali nelle card
- simboli grandi in box premio o sfida

--dimensione-icona-piccola
Valore: clamp(1rem, 1.8vw, 1.25rem)

- simboli secondari nelle card

--altezza-minima-toolbar

Possibili utilizzi:
- altezza minima della navbar

- barra superiore responsive
- header principale del sito
- contenitore navigazione superiore






Indicazione generale:
- Usa le variabili di spaziatura per mantenere uniformi padding, margin e gap in tutto il sito.
- Usa i raggi bordo in base all’importanza del contenitore: piccolo per controlli, medio per card standard, grande per box principali.
- Usa le variabili font per mantenere una tipografia coerente e responsive.
- Usa le variabili di dimensione per controlli, avatar e icone in modo uniforme tra tutte le pagine.
